Bowery's Kryptonite

Buttons

Buttons should only be used when the resulting action does not cause a page load. Otherwise you should use an a tag.

Code


$("a[href='Degrassi']").click(function() {
  $("html, body").animate({ scrollTop: 0 }, "slow");
  alert("Started from the bottom, now we're here");
  return false;
});
